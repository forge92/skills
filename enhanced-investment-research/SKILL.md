# 增强版投资研究

## 概述

本技能旨在通过整合多位价值投资大师（如巴菲特、芒格、段永平、李录）的智慧，并结合现代科学投资分析方法论（如因果推理、量化与质性分析结合），构建一个更科学、更全面的投资研究框架。它利用多 Agent 协作模式，模拟专业投研团队的工作流程，提供深度、严谨且可解释的投资分析报告，帮助用户做出更明智的投资决策。

## 核心能力

### 1. 深度公司研究 (Deep Company Research)

- **多视角综合分析**：从商业模式、财务估值、竞争格局、管理层、长期确定性等多个维度，结合四位投资大师的视角进行全面评估。
- **因果推理与护城河分析**：深入挖掘公司业务的本质和驱动因素，识别并评估其竞争优势和护城河。
- **结构化反偏见机制**：内置信息丰富度评级、芒格式逆向检验、快速否决清单等机制，减少认知偏差。

### 2. 严谨财务分析 (Rigorous Financial Analysis)

- **关键数据交叉验证**：对市值、估值指标等关键财务数据进行多源交叉验证，并使用精确计算工具（如 Python 的 `decimal` 模块）确保准确性。
- **多情景估值模型**：提供乐观、中性、悲观情景下的估值分析，并进行反向 DCF 估值。
- **财报精读与解读**：深入分析公司财报，结合业务发展进行解读，而非简单罗列数据。

### 3. 行业与市场筛选 (Industry & Market Screening)

- **产业链全景扫描**：从投资主题出发，构建产业链逻辑链，识别上下游投资机会。
- **行业漏斗筛选**：通过多阶段筛选（全市场 → 粗筛 → 精细分析 → 终选），从海量公司中精选优质标的。
- **质量筛选与瓶颈猎手**：运用硬性指标快速排除非一流公司，并识别产业链中的物理瓶颈和套利机会。

### 4. 持仓与风险管理 (Portfolio & Risk Management)

- **投资论文追踪**：买入后持续跟踪投资逻辑是否被证伪，及时调整策略。
- **组合管理与优化**：提供仓位、集中度、再平衡等组合管理建议。
- **股价异动归因**：快速分析股价异常波动的原因。

### 5. 投资思维工具 (Investment Thought Tools)

- **巴菲特买入前 Checklist**：六关快速筛选，帮助用户在短时间内判断公司是否值得深入研究。
- **段永平问答模式**：以段永平的商业洞察力思考任何商业或投资问题。
- **可复现的研究流程**：确保每次分析输出的结构、深度和覆盖面一致，便于横向对比和时间序列分析。

## 工作流程示例

### 1. 深度公司研究工作流

1.  **输入**：用户提供公司名称或股票代码。
2.  **数据收集**：通过 `stock-analysis` 技能获取公司概况、财务数据、SEC 文件等基础信息。
3.  **数据验证**：使用 `scripts/data_validator.py` 对关键财务数据进行交叉验证，确保数据准确性。
4.  **多 Agent 分析**：
    - **商业模式 Agent (段永平视角)**：分析公司生意本质、护城河、竞争优势。参考 `references/investment_masters_principles.md`。
    - **财务估值 Agent (巴菲特视角)**：进行财务数据分析、估值计算、安全边际评估。使用 `scripts/financial_calculator.py` 进行精确计算，参考 `references/valuation_models.md`。
    - **风险与竞争 Agent (芒格视角)**：进行逆向思考，识别潜在风险、竞争威胁。使用 `scripts/anti_bias_check.py` 中的 `perform_reverse_thinking` 函数。
    - **长期确定性 Agent (李录视角)**：评估公司长期发展趋势、管理层文化、文明趋势。参考 `references/investment_masters_principles.md`。
5.  **反偏见检查**：使用 `scripts/anti_bias_check.py` 中的 `check_quick_veto` 和 `assess_information_richness` 函数进行快速否决和信息丰富度评级。
6.  **Team Lead 综合**：汇总各 Agent 报告，进行交叉验证，生成综合决策备忘录和投资建议。
7.  **输出**：使用 `templates/research_report_template.md` 生成结构化的投资研究报告，包含结论、评分、风险提示和建议。

### 2. 行业筛选工作流

1.  **输入**：用户提供行业或投资主题。
2.  **产业链分析**：构建投资逻辑链，绘制产业链全景图。参考 `references/industry_analysis_frameworks.md`。
3.  **市场扫描与粗筛**：根据预设指标（如活跃度、涨幅、市值）从全市场筛选出初步标的。
4.  **精细分析与终选**：对粗筛后的公司进行详细分析，结合组合互补性选出最终推荐标的。
5.  **输出**：行业研究报告，包含产业链分析、推荐组合及操作信号。

## 资源

本技能包含以下辅助资源：

### `scripts/`

- `financial_calculator.py`：用于精确计算关键财务指标（如 DCF、PE、PB、市值校验）。
- `anti_bias_check.py`：实现反偏见机制的脚本，如芒格式逆向检验、快速否决清单。
- `data_validator.py`：用于多源数据交叉验证的脚本。

### `references/`

- `investment_masters_principles.md`：详细阐述巴菲特、芒格、段永平、李录的投资哲学和分析框架。
- `valuation_models.md`：各种估值模型的理论与实践指南。
- `industry_analysis_frameworks.md`：行业分析的理论框架和方法。

### `templates/`

- `research_report_template.md`：结构化投资研究报告的 Markdown 模板。
- `checklist_template.md`：巴菲特买入前 Checklist 的模板。
