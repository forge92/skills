# 巴菲特买入前 Checklist

## 公司名称：{{COMPANY_NAME}}

### 第一关：能力圈（我能理解吗？）

- [ ] 我能清晰地解释这家公司的产品、服务和商业模式。
- [ ] 我理解这家公司如何赚钱，以及其收入和利润的驱动因素。
- [ ] 我对这家公司所处的行业有基本的了解。

**结论**：[通过/不通过/需要更多研究]

### 第二关：好生意（经济特征如何？）

- [ ] 这家公司是否拥有持续的盈利能力？
- [ ] 这家公司的毛利率和净利率是否健康且稳定？
- [ ] 这家公司是否需要大量的资本投入来维持运营和增长？
- [ ] 这家公司是否能产生稳定的自由现金流？

**结论**：[通过/不通过/需要更多研究]

### 第三关：护城河（竞争优势深不深？）

- [ ] 这家公司是否拥有强大的品牌优势？
- [ ] 这家公司是否拥有成本优势（如规模经济、独特生产技术）？
- [ ] 客户转换到竞争对手的成本是否很高？
- [ ] 这家公司是否受益于网络效应（用户越多，价值越大）？
- [ ] 这家公司是否拥有难以复制的专利、技术或特许经营权？

**结论**：[通过/不通过/需要更多研究]

### 第四关：管理层（值得信任吗？）

- [ ] 管理层是否诚信、正直，并以股东利益为重？
- [ ] 管理层是否具备卓越的经营能力和战略眼光？
- [ ] 管理层是否能够有效地配置资本？
- [ ] 管理层是否坦诚地面对问题和挑战？

**结论**：[通过/不通过/需要更多研究]

### 第五关：安全边际（价格便宜吗？）

- [ ] 当前股价是否显著低于我对公司内在价值的估计？
- [ ] 即使我对未来预测出现偏差，当前价格是否仍能提供足够的保护？
- [ ] 公司的负债水平是否合理，财务结构是否稳健？

**结论**：[通过/不通过/需要更多研究]

### 第六关：决策纪律（是理性还是FOMO？）

- [ ] 我的投资决策是基于理性的分析，而不是受市场情绪或“害怕错过”的影响？
- [ ] 我是否愿意在市场低迷时买入，并在市场狂热时保持冷静？
- [ ] 我是否做好了长期持有的准备，并能承受短期波动？

**结论**：[通过/不通过/需要更多研究]
