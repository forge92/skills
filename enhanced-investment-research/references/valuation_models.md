# 估值模型理论与实践

## 1. 现金流折现模型 (Discounted Cash Flow, DCF)

### 理论基础

DCF 模型是价值投资中最核心的估值方法之一，其基本思想是：一个资产的内在价值等于其未来所能产生的自由现金流的折现值之和。它基于“时间价值”原理，认为未来的现金流不如现在的现金流有价值。

### 计算步骤

1.  **预测自由现金流 (Free Cash Flow, FCF)**：预测公司未来 5-10 年的自由现金流。FCF 通常定义为：
    `FCF = 净利润 + 折旧与摊销 - 资本支出 - 营运资本变动`
    或
    `FCF = 息税前利润 (EBIT) * (1 - 税率) + 折旧与摊销 - 资本支出 - 营运资本变动`
2.  **确定折现率 (Discount Rate)**：通常使用加权平均资本成本 (Weighted Average Cost of Capital, WACC)。
    `WACC = (E/V) * Re + (D/V) * Rd * (1 - Tc)`
    其中：
    - `E`：公司股权的市场价值
    - `D`：公司债务的市场价值
    - `V = E + D`：公司总市场价值
    - `Re`：股权成本 (通常使用资本资产定价模型 CAPM 计算)
    - `Rd`：债务成本
    - `Tc`：企业所得税率
3.  **计算终值 (Terminal Value, TV)**：预测期结束后，公司通常会继续经营。终值代表预测期之后所有自由现金流的折现值。常用的计算方法有：
    - **永续增长模型 (Gordon Growth Model)**：`TV = FCFn * (1 + g) / (WACC - g)`
        其中 `FCFn` 是预测期最后一年的自由现金流，`g` 是永续增长率。
    - **退出市盈率法 (Exit Multiple Method)**：`TV = 预测期末 EBITDA * 行业平均 EBITDA 倍数`
4.  **计算内在价值**：将预测期内的自由现金流和终值折现到当前，求和即得到公司内在价值。

### 优点与缺点

- **优点**：理论严谨，能够反映企业长期价值，对管理层决策有指导意义。
- **缺点**：对未来现金流、折现率和永续增长率的假设敏感，预测难度大，容易受主观判断影响。

## 2. 相对估值法 (Relative Valuation)

### 理论基础

相对估值法通过将目标公司与可比公司或行业平均水平进行比较，来确定其价值。它假设市场对类似资产的定价是有效的。

### 常用指标

- **市盈率 (Price-to-Earnings Ratio, P/E)**：`P/E = 股价 / 每股收益` 或 `P/E = 市值 / 净利润`
    - 适用于盈利稳定且有可比公司的行业。
- **市净率 (Price-to-Book Ratio, P/B)**：`P/B = 股价 / 每股净资产` 或 `P/B = 市值 / 股东权益`
    - 适用于资产密集型行业，如金融、房地产。
- **市销率 (Price-to-Sales Ratio, P/S)**：`P/S = 股价 / 每股销售收入` 或 `P/S = 市值 / 销售收入`
    - 适用于初创公司或亏损公司，因为销售收入通常比利润更稳定。
- **企业价值/息税折旧摊销前利润 (EV/EBITDA)**：`EV/EBITDA = (市值 + 净债务) / EBITDA`
    - 适用于资本结构差异较大的公司，或需要考虑非现金支出的行业。

### 优点与缺点

- **优点**：简单易懂，易于操作，能够反映市场情绪和行业估值水平。
- **缺点**：难以找到完全可比的公司，市场情绪可能导致估值偏差，不适用于独特商业模式的公司。

## 3. 资产基础估值法 (Asset-Based Valuation)

### 理论基础

资产基础估值法通过评估公司资产的价值来确定其整体价值。它通常用于重资产行业、清算估值或作为其他估值方法的补充。

### 常用方法

- **账面价值法**：直接使用公司资产负债表上的账面价值。
- **重置成本法**：估算重新建造或购买公司所有资产所需的成本。
- **清算价值法**：估算公司在清算时，出售所有资产所能获得的净值。

### 优点与缺点

- **优点**：相对客观，适用于重资产公司和清算场景。
- **缺点**：难以准确评估无形资产价值，账面价值可能无法反映市场价值，不适用于轻资产或高成长公司。

## 4. 期权估值模型 (Option Pricing Models)

### 理论基础

期权估值模型（如 Black-Scholes 模型）主要用于评估金融期权、可转换债券或具有期权特征的资产。在某些情况下，也可以用于评估公司股权的期权价值，尤其是在公司面临破产风险时。

### 优点与缺点

- **优点**：适用于具有期权特征的复杂金融产品。
- **缺点**：模型假设严格，参数难以准确估计，不适用于传统股权估值。

## 估值方法的选择与结合

- **没有万能的估值方法**：不同的估值方法适用于不同的公司和行业。
- **多种方法结合**：通常建议结合使用多种估值方法，例如 DCF 估值作为核心，辅以相对估值法进行校准，以获得更全面的价值区间。
- **敏感性分析**：对关键假设（如增长率、折现率、倍数）进行敏感性分析，了解估值结果的波动范围。
- **反向 DCF**：根据当前股价反推市场隐含的增长率或回报率，判断市场预期是否合理。
